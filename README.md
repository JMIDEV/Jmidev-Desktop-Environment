# JMIDEV Desktop Environment

A desktop environment/program based and focused on web technologies and nodejs. Runs using NW.js

Here's a video demonstration on YouTube, featuring a full working window system with tiling, a working taskbar, a start menu from which you can launch apps, a Settings app that allows the user to change the wallpaper and check some system specifications; a Calculator app inspired by the Casio fx-82MS, and an unfinished file explorer (you must specify the route to navigate from the code itself).

This project was left unfinished back in 2023.
